Apri un file di dati da un indirizzo Internet

Utilizzare questa casella di dialogo per aprire un file di dati in formato SPSS, Excel, SAS o Stata dal Web. Il file verrà scaricato sul computer e quindi aperto. La finestra di dialogo può anche essere utilizzata per scaricare e salvare i file di qualsiasi altro tipo, ma questi non verranno aperti.

Nome file di dati e ubicazione Web. Immettere l'indirizzo Internet completo di un file da aprire, incluso il protocollo, ad esempio http:// o ftp://. Il file verrà scaricato in una cartella temporanea sul computer e aperto. Salvarlo in una ubicazione permanente se si prevede di utilizzarlo di nuovo.

Tipo di file. Scegliere SPSS, Excel, SAS o Stata come tipo di file. Le versioni di questi file che sono supportate corrispondono alle versioni che è possibile aprire dalla casella di dialogo standard visualizzata tramite File->Apri. Scegliere Altro, solo per scaricare e salvare un file.

Salva cartella. Se si seleziona Altro come tipo di file, specificare una cartella esistente in cui il file deve essere memorizzato. Il nome del file verrà preso dall'uri di download.

Un file esistente può essere o meno sovrascritto a seconda del sistema operativo e delle autorizzazioni al file. Se si verifica un conflitto di nomi file, il file può essere lasciato nella cartella di destinazione con un nome temporaneo.

Nome dataset da assegnare. Facoltativamente, assegnare un nome dataset SPSS al file da aprire. Questo non è un nome file, ma un nome utilizzato per gestire i file di dati quando è possibile aprirne più di uno.

Opzioni per file Excel. Queste opzioni sono valide solo per file Excel e vengono ignorate se viene utilizzato un altro tipo di file. Corrispondono alle opzioni disponibili dalla finestra di dialogo standard visualizzata tramite File->Apri.

Numero foglio da leggere. Se il file ha più di un foglio, specificare quale leggere in base al numero o al nome. Per impostazione predefinita, viene letto il foglio numero 1. Non specificare un nome e un numero insieme.

Nome foglio da leggere. È possibile specificare il foglio da leggere immettendone il nome qui. Se vengono forniti sia il nome che il numero il nome ha la priorità.

Intervallo celle. Per impostazione predefinita, vengono lette tutte le celle. È possibile specificare un intervallo di celle per limitare l'area letta.

Leggi i nomi delle variabili dalla prima riga. Selezionare questa casella per prendere i nomi delle variabili per il dataset SPSS dalla prima riga dell'intervallo. I nomi che non sono validi come nomi SPSS verranno sostituiti automaticamente.

Presunta larghezza stringa massima. Per impostazione predefinita, alle stringhe è assegnata una larghezza di 32767. È possibile specificare un numero inferiore qui.

Opzioni per file SAS. È possibile immettere il nome di un dataset SAS specifico all'interno del file da leggere.

Funzioni aggiuntive

Eseguire il comando
SPSSINC GETURI DATA /HELP.
per visualizzare la descrizione della sintassi completa.

Requisiti

Questo comando richiede Python Essentials. È possibile scaricarlo tramite il sito Web SPSS Community all'indirizzo www.ibm.com/developerworks/spssdevcentral o reperirlo dal materiale di installazione. Essentials viene installato automaticamente con Statistics 22 o successive.
