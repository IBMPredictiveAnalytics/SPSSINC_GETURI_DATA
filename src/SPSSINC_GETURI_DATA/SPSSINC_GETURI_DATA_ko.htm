인터넷 주소에서 데이터 파일 열기

이 대화 상자를 사용하여 웹에서 SPSS, Excel, SAS 또는 Stata 형식의 데이터 파일을 열 수 있습니다. 파일이 컴퓨터에 다운로드된 후 열립니다. 또한 이 대화 상자를 사용하여 다른 유형의 파일을 다운로드하고 저장할 수 있습니다. 단, 이러한 파일은 열리지 않습니다.

웹 위치 및 데이터 파일 이름. http:// 또는 ftp:// 등의 프로토콜을 비롯하여 열려는 파일의 전체 인터넷 주소를 입력하십시오. 파일이 컴퓨터의 임시 디렉토리에 다운로드된 후 열립니다. 파일을 다시 사용하려면 영구 위치에 저장하십시오.

파일 유형. SPSS, Excel, SAS 또는 Stata를 파일 유형으로 선택하십시오. 지원되는 이러한 파일의 버전은 표준 파일 열기 대화 상자에서 열 수 있는 버전에 해당합니다. 파일을 다운로드하고 저장하려면 기타를 선택하십시오.

폴더 저장. 기타를 파일 유형으로 선택할 경우 파일을 저장할 기존 폴더를 지정하십시오. 파일 이름은 다운르도 URI에서 가져옵니다.

기존 파일은 운영 체제 및 파일 권한에 따라 덮어쓸 수 있습니다. 파일 이름 충돌이 발생할 경우 파일이 대상 폴더에 임시 이름으로 남아 있을 수 있습니다.

지정할 데이터 세트 이름. 필요한 경우 열려는 파일에 대한 SPSS 데이터 세트 이름을 지정하십시오. 이것은 파일 이름은 아니지만 이 이름은 둘 이상의 데이터 파일을 열 때 해당 파일을 관리하는 데 사용됩니다.

Excel 파일 옵션. 이 옵션은 Excel 파일에만 적용되며 다른 파일 유형이 사용될 경우 무시됩니다. 이 옵션은 표준 파일 열기 대화 상자에서 제공되는 옵션에 해당합니다.

읽을 시트 번호. 파일에 둘 이상의 시트가 있으면 번호 또는 이름별로 읽을 시트를 지정하십시오. 기본적으로 시트 번호 1이 읽힙니다. 이름 및 번호를 모두 지정하지 마십시오.

읽을 시트 이름. 여기에 이름을 입력하여 읽을 시트를 지정할 수 있습니다. 이름과 번호가 모두 지정된 경우 이름이 우선합니다.

셀 범위. 기본적으로 모든 셀이 읽힙니다. 읽을 영역을 제한하는 셀 범위를 지정하십시오.

첫 번째 행에서 변수 이름 읽기. 범위의 첫 번째 행에서 SPSS 데이터 세트에 대한 변수 이름을 가져오려면 이 상자를 선택하십시오. SPSS 이름으로 올바르지 않은 이름은 자동으로 대체됩니다.

추정된 최대 문자열 너비. 기본적으로 문자열에 32767의 너비가 지정됩니다. 여기에 더 작은 숫자를 지정할 수 있습니다.

SAS 파일 옵션. 읽을 파일 내에서 특정 SAS 데이터 세트의 이름을 입력할 수 있습니다.

추가 기능

다음 명령문을 실행하십시오.
SPSSINC GETURI DATA /HELP.
전체 명령문 설명을 표시합니다.

요구 사항

이 명령문에는 Python Essentials가 필요합니다. Python Essentials는 www.ibm.com/developerworks/spssdevcentral의 SPSS Community 웹 사이트 또는 설치 자료에서 다운로드할 수 있습니다. Essentials는 Statistics 22 이상에 자동으로 설치됩니다.
