Datendatei über eine Internetadresse öffnen

Über dieses Dialogfeld kann eine Datendatei im SPSS-, Excel-, SAS- oder Stata-Format über das Web geöffnet werden. Die Datei wird auf Ihren Computer heruntergeladen und anschließend geöffnet. Darüber hinaus können über das Dialogfeld Dateien jeglichen anderen Typs heruntergeladen und gespeichert werden; diese werden jedoch nicht geöffnet.

Webstandort und Name der Datendatei. Geben Sie die vollständige Internetadresse der zu öffnenden Datei einschließlich des Protokolls (z. B. http:// oder ftp://) ein. Die Datei wird in ein temporäres Verzeichnis auf Ihrem Computer heruntergeladen und geöffnet. Wenn Sie die Datei ein weiteres Mal verwenden wollen, speichern Sie sie an einem permanenten Speicherort.

Dateityp. Wählen Sie SPSS, Excel, SAS oder Stata als Dateityp aus. Für diese Dateien werden dieselben Versionen unterstützt, die auch über das Standarddialogfeld zum Öffnen von Dateien geöffnet werden können. Wählen Sie 'Andere' aus, wenn eine Datei nur heruntergeladen und gespeichert werden soll.

Ordner speichern. Wenn Sie 'Andere' als Dateityp auswählen, geben Sie einen vorhandenen Ordner an, in dem die Datei gespeichert werden soll. Der Dateiname wird dem Download-URI entnommen.

Ob eine bereits vorhandene Datei überschrieben wird, hängt von den Betriebssystem- und Dateiberechtigungen ab. Wenn ein Dateinamenskonflikt auftritt, wird die Datei möglicherweise im Zielordner unter einem temporären Namen abgelegt.

Zuzuweisender Datasetname. Optional können Sie der Datei, die geöffnet wird, einen SPSS-Datasetnamen zuweisen. Dabei handelt es sich nicht um einen Dateinamen, sondern um einen Namen, der zur Verwaltung von Datendateien verwendet wird, wenn mehrere geöffnet sind.

Optionen für Excel-Dateien. Diese Optionen sind nur bei Excel-Dateien anwendbar und werden ignoriert, wenn ein anderer Dateityp verwendet wird. Sie entsprechen den Optionen, die über das Standarddialogfeld zum Öffnen von Dateien verfügbar sind.

Nummer des zu lesenden Arbeitsblatts. Wenn die Datei mehr als ein Arbeitsblatt hat, geben Sie die Nummer oder den Namen des Arbeitsblatts an, das gelesen werden soll. Standardmäßig wird das Arbeitsblatt mit der Nummer 1 gelesen. Geben Sie nicht beides, Namen und Nummer, gleichzeitig an.

Name des zu lesenden Arbeitsblatts. Hier können Sie den Namen des Arbeitsblatts eingeben, das gelesen werden soll. Werden Name und Nummer angegeben, hat der Name Priorität.

Zellenbereich. Standardmäßig werden alle Zellen gelesen. Durch Angabe eines Zellenbereichs kann der zu lesende Bereich begrenzt werden.

Variablennamen aus erster Zeile lesen. Wählen Sie dieses Feld aus, um die Variablennamen für das SPSS-Dataset aus der ersten Zeile des Bereichs zu verwenden. Namen, die keine gültigen SPSS-Namen sind, werden automatisch ersetzt.

Angenommene maximale Zeichenfolgenbreite. Zeichenfolgen wird standardmäßig eine Breite von 32767 zugewiesen. Hier kann ein kleinerer Wert angegeben werden.

Optionen für SAS-Dateien. Hier kann der Name eines bestimmten SAS-Datasets in der zu lesenden Datei eingegeben werden.

Weitere Funktionen

Führen Sie den folgenden Befehl aus:
SPSSINC GETURI DATA /HELP.
Sie zeigen damit die vollständige Syntaxbeschreibung an.

Anforderungen

Für diesen Befehl ist Python Essentials erforderlich. Sie können Python Essentials von der SPSS Community-Website (www.ibm.com/developerworks/spssdevcentral) oder über Ihr Installationsmaterial herunterladen. Ab Statistics 22 wird Python Essentials automatisch installiert.
