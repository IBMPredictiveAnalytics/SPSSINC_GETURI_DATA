Abrir un archivo de datos desde una dirección de Internet

Utilice este cuadro de diálogo para abrir un archivo de datos en formato SPSS, Excel, SAS o Stata desde la Web. El archivo se descargará en el equipo y se abrirá. El diálogo también se puede utilizar para descargar y guardar archivos de cualquier otro tipo, pero no se abrirán.

Ubicación web y nombre de archivo de datos. Especifique la dirección completa de Internet de un archivo que vaya a abrir, incluido el protocolo http:// o ftp://. El archivo se descargará en un directorio temporal del equipo y se abrirá. Guárdelo en una ubicación permanente si tiene previsto volver a utilizarlo.

Tipo de archivo. Seleccione SPSS, Excel, SAS o Stata como tipo de archivo. Las versiones de estos archivos que se admiten corresponden a las versiones que se pueden abrir desde el cuadro de diálogo Abrir archivo estándar. Elija Otros si sólo desea descargar y guardar un archivo.

Carpeta de guardado. Si selecciona Otros como tipo de archivo, especifique una carpeta existente donde debe guardarse el archivo. El nombre del archivo se tomará del URI descargado.

Un archivo existente puede sobrescribirse o no en función del sistema operativo y de los permisos de archivos. Si se produce un conflicto de nombres de archivos, el archivo puede guardarse en la carpeta de destino con un nombre temporal.

Nombre de conjunto de datos que se va a asignar. Asigna opcionalmente un nombre de conjunto de datos SPSS al archivo que se va a abrir. No se trata de un nombre de archivo sino de un nombre que se utiliza para gestionar archivos de datos cuando se puede abrir más de uno.

Opciones para archivos Excel. Estas opciones se aplican sólo a los archivos Excel y se pasan por alto si se utiliza otro tipo de archivo. Corresponden a las opciones disponibles del cuadro de diálogo Abrir archivo estándar.

Número de la hoja que se va a leer. Si el archivo tiene más de una hoja, especifique cuál se debe leer mediante la especificación de un número o nombre. De manera predeterminada, se lee la hoja número 1. No especifique los dos, el nombre y el número.

Nombre de la hoja que se va a leer. Puede especificar la hoja que se debe leer mediante la especificación del nombre. Si se especifican tanto el nombre como el número, el nombre tiene prioridad.

Rango de casillas. De manera predeterminada, se leen todas las casillas. Puede especificar un rango de casillas para limitar el área de lectura.

Leer nombre de variables de la primera fila. Marque esta casilla para tomar los nombres de variables del conjunto de datos SPSS a partir de la primera fila del rango. Los nombres que no son válidos como nombres de SPSS se sustituirán automáticamente.

Longitud máxima de cadena supuesta. De manera predeterminada, las cadenas tienen una longitud de 32767. Puede especificar un número más pequeño.

Opciones para archivos SAS. Puede especificar el nombre de un conjunto de datos SAS específico dentro del archivo que se va a leer.

Características adicionales

Ejecute el comando
SPSSINC GETURI DATA /HELP.
para mostrar toda la descripción de la sintaxis.

Requisitos

Este comando requiere Python Essentials. Puede descargarlo a través del sitio web de la Comunidad de SPSS en www.ibm.com/developerworks/spssdevcentral o de los materiales de instalación. Essentials se instala automáticamente con Statistics 22 o posterior.
