Ouverture d'un fichier de données à partir d'une adresse Internet

Cette boîte de dialogue permet d'ouvrir un fichier de données SPSS, Excel, SAS ou Stata à partir du Web. Le fichier est téléchargé sur votre ordinateur, puis il est ouvert. Vous pouvez également utiliser cette boîte de dialogue pour télécharger et enregistrer des fichiers d'un type différent, mais ceux-ci ne seront pas ouverts.

Emplacement Web et nom de fichier de données : Entrez l'adresse complète du fichier à ouvrir, sans oublier le protocole http:// ou ftp://. Le fichier est téléchargé vers un répertoire temporaire de votre ordinateur, puis il est ouvert. Enregistrez-le dans un emplacement permanent si vous prévoyez de l'utiliser ultérieurement.

Type de fichier : Choisissez un type de fichier SPSS, Excel, SAS ou Stata. Les versions prises en charge correspondent à celles à partir desquelles vous pouvez ouvrir les fichiers dans la boîte de dialogue standard Fichier - Ouvrir. Choisissez le type Autre pour télécharger et enregistrer un fichier uniquement.

Dossier d'enregistrement : Si vous choisissez le type de fichier Autre, vous devez définir le dossier existant dans lequel stocker le fichier. Le nom de fichier est établi à partir de l'URI de téléchargement.

Un fichier existant peut être écrasé ou non en fonction du système d'exploitation et des droits d'accès au fichier. Si une collision de nom est détectée, il se peut que le fichier reste dans le dossier cible avec un nom temporaire.

Nom de fichier à attribuer : Si vous le souhaitez, vous pouvez affecter un nom de fichier SPSS au fichier ouvert. Il ne s'agit pas du nom du fichier, mais du nom servant à gérer les fichiers de données lorsque plusieurs fichiers sont ouverts.

Options des fichiers Excel : Ces options s'appliquent uniquement aux fichiers Excel et sont ignorées lorsqu'un autre type de fichier est utilisé. Elles correspondent aux options disponibles dans la boîte de dialogue standard d'ouverture de fichier.

Numéro de feuille à lire : Si le fichier comporte plusieurs feuilles, indiquez le numéro ou le nom de la feuille que vous voulez lire. Par défaut, le numéro de feuille 1 est lu. Indiquez le nom ou le numéro de la feuille.

Nom de feuille à lire : Vous pouvez indiquer la feuille que vous voulez lire en entrant ici son nom. Si un nom et un numéro sont définis, le nom est prioritaire.

Plage de cellules : Par défaut, toutes les cellules sont lues. Vous pouvez choisir d'indiquer une plage de cellules afin de limiter la zone de lecture.

Lire les noms de variable à partir de la première ligne : Cochez cette case pour lire les noms de variable du jeu de données SPSS à partir de la première ligne de la plage. Les noms qui ne sont pas valides en tant que noms SPSS sont automatiquement remplacés.

Largeur de chaîne maximale prise en compte : Par défaut, les chaînes ont une largeur de 32 767. Vous pouvez indiquer une valeur inférieure ici.

Options des fichiers SAS : Vous pouvez entrer le nom d'un jeu de données SAS spécifique dans le fichier à lire.

Fonctions supplémentaires

Exécutez la commande
SPSSINC GETURI DATA /HELP.
pour afficher la description intégrale de la syntaxe.

Conditions requises

Cette commande requiert Python Essentials. Vous pouvez le télécharger depuis le site Web de la communauté SPSS à l'adresse www.ibm.com/developerworks/spssdevcentral ou à partir de vos supports d'installation. Essentials est automatiquement installé avec Statistics 22 ou version ultérieure.
