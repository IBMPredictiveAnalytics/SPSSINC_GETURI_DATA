Abrir um arquivo de dados a partir de um endereço na Internet

Use esta caixa de diálogo para abrir um arquivo de dados no formato SPSS, Excel, SAS ou Stata a partir da web. O arquivo será transferido por download para seu computador e, em seguida, aberto. O diálogo também pode ser usado para fazer download e salvar arquivos de qualquer outro tipo, mas esses não serão abertos.

Localização na web e nome do arquivo de dados. Insira o endereço na Internet completo de um arquivo para abrir, incluindo o protocolo, como http:// ou ftp://. O arquivo será transferido por download para um diretório temporário em seu computador e aberto. Salve-o em uma localização permanente se planeja usá-lo novamente.

Tipo de arquivo. Escolha SPSS, Excel, SAS ou Stata como o tipo de arquivo. As versões desses arquivos que são suportados correspondem às versões que você pode abrir a partir da caixa de diálogo padrão Abrir arquivo. Escolha Outro para apenas fazer download e salvar um arquivo.

Salvar pasta. Se você selecionar Outro como o tipo de arquivo, especifique uma pasta existente onde o arquivo deve ser armazenado. O nome do arquivo será obtido do uri de download.

Um arquivo existente pode ou não pode ser sobrescrito, dependendo do sistema operacional e das permissões do arquivo. Se ocorrer uma colisão de nomes de arquivo, o arquivo pode ser deixado na pasta de destino com um nome temporário.

Nome do conjunto de dados a designar. Opcionalmente, designe um nome do conjunto de dados do SPSS para o arquivo que está sendo aberto. Este não é um nome de arquivo, mas um nome usado para gerenciar arquivos de dados quando mais de um pode ser aberto.

Opções para arquivos do Excel. Estas opções se aplicam apenas aos arquivos do Excel e são ignoradas se outro tipo de arquivo for usado. Elas correspondem às opções disponíveis do diálogo padrão Abrir arquivo.

Número da planilha para ler. Se o arquivo tiver mais de uma planilha, especifique qual ler pelo número ou nome. Por padrão, a planilha número 1 é lida. Não especifique ambos um nome e um número.

Nome da planilha para ler. É possível especificar a planilha para ler inserindo o nome aqui. Se ambos um nome e um número forem fornecidos, o nome terá prioridade.

Intervalo de células. Por padrão, todas as células são lidas. É possível especificar um intervalo de células para limitar a área lida.

Ler nomes de variáveis a partir da primeira linha. Marque esta caixa para obter os nomes de variáveis para o conjunto de dados do SPSS da primeira linha do intervalo. Nomes que são inválidos, como nomes do SPSS, serão automaticamente substituídos.

Largura máxima da sequência de caracteres assumida. Por padrão, as sequências de caracteres têm uma largura de 32767 designada. É possível especificar um número menor aqui.

Opções para arquivos SAS. É possível inserir o nome de um conjunto de dados SAS específico dentro do arquivo para ler.

Recursos adicionais

Execute o comando
SPSSINC GETURI DATA /HELP.
para exibir a descrição toda da sintaxe.

Requisitos

Este comando requer o Python Essentials. É possível fazer download dele por meio do website SPSS Community em www.ibm.com/developerworks/spssdevcentral ou de seus materiais de instalação. O Essentials é instalado automaticamente com o Statistics 22 ou posterior.
