インターネット アドレスからデータ ファイルを開く

このダイアログ ボックスを使用して、SPSS、Excel、SAS、または Stata 形式のデータ ファイルを Web から開きます。 ファイルは、ご使用のコンピュータにダウンロードされてから、開きます。このダイアログは、他のタイプのファイルをダウンロードして保存する場合にも使用できますが、それらのファイルは開きません。

Web ロケーションおよびデータ ファイル名。 開くファイルの完全なインターネット アドレスを入力します (http:// や ftp:// などのプロトコルを含みます)。 ファイルは、ご使用のコンピュータ上の一時ディレクトリにダウンロードされ、開きます。ファイルを再使用する予定の場合は、永続的な場所に保存してください。

ファイル タイプ。 ファイル タイプとして「SPSS」、「Excel」、「SAS」、または「Stata」を選択します。これらのファイルでサポートされるバージョンは、標準の「ファイル」>「開く」ダイアログ ボックスから開くことができるバージョンに対応します。ファイルのダウンロードと保存のみを行う場合は、「その他」を選択してください。

フォルダの保存。ファイル タイプとして「その他」を選択した場合は、ファイルが保管される既存のフォルダを指定します。ファイル名は、ダウンロード URI から取得されます。

既存のファイルが上書きされるかどうかは、オペレーティング システムやファイル許可によって異なります。ファイル名の衝突が発生した場合、ファイルはターゲット フォルダ内に一時的な名前で残される可能性があります。

割り当てるデータセット名。 必要に応じて、開く予定のファイルに SPSS データセット名を割り当てます。これは、ファイル名ではなく、複数のファイルが開いているときにデータ ファイルを管理するのに使用される名前です。

Excel ファイルのオプション。 これらのオプションは Excel ファイルにのみ適用され、他のファイル タイプを使用する場合は無視されます。これらは、標準の「ファイル」>「開く」ダイアログから使用できるオプションに対応します。

読み取るシート番号。 ファイルに複数のシートがある場合は、読み取るシートを番号または名前で指定します。デフォルトでは、シート番号 1 が読み取られます。名前と番号の両方を指定しないでください。

読み取るシート名。 読み取るシートを指定するには、ここに名前を入力します。名前と番号の両方が指定された場合は、名前が優先されます。

セル範囲。デフォルトでは、すべてのセルが読み取られます。セル範囲を指定すると、読み取る領域を制限できます。

最初の行から変数名を読み取る。 このボックスにチェック マークを付けると、この範囲の最初の行から SPSS データセットの変数名が取得されます。 SPSS 名として無効な名前は、自動的に置き換えられます。

想定される最大ストリング幅。 デフォルトでは、ストリングには幅 32767 が割り当てられます。それよりも小さい数値をここで指定できます。

SAS ファイルのオプション。 読み取るファイル内の特定 SAS データセットの名前を入力できます。

その他の機能

次のコマンドを実行します。
SPSSINC GETURI DATA /HELP.
シンタックス全体の説明が表示されます。

要件

このコマンドには Python Essentials が必要です。 これらは、SPSS Community Web サイト (www.ibm.com/developerworks/spssdevcentral) またはインストール素材からダウンロードできます。Essentials は、Statistics 22 以降のバージョンと共に自動的にインストールされます。
