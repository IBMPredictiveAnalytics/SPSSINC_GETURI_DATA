從網際網路位址開啟資料檔

使用此對話框從 Web 開啟 SPSS、Excel、SAS 或 Stata 格式的資料檔。 此檔案會下載至您的電腦，然後開啟。此對話框也可以用於下載及儲存任何其他類型的檔案，但不會開啟這些檔案。

Web 位置及資料檔名。 輸入要開啟的檔案的完整網際網路位址，包括通訊協定，如 http:// 或 ftp://。 此檔案會下載至您電腦上的暫存目錄並開啟。如果您計劃再次使用它，請將它儲存在永久位置。

檔案類型。 選擇 SPSS、Excel、SAS 或 Stata 作為檔案類型。支援的這些檔案版本對應於您可以從標準「開啟檔案」對話框開啟的版本。選擇「其他」，以僅下載並儲存檔案。

儲存資料夾。 如果選取「其他」作為檔案類型，請指定應在其中儲存檔案的現有資料夾。將從下載 URI 取得檔名。

視作業系統及檔案權限而定，可能會，也可能不會改寫現有檔案。如果檔名發生衝突，檔案可能會以暫存名稱保留在目標資料夾中。

要指派的資料集名稱。 選擇性地將 SPSS 資料集名稱指派給要開啟的檔案。這不是檔名，而是在可能開啟多個資料檔時用於管理資料檔的名稱。

用於 Excel 檔案的選項。 這些選項僅適用於 Excel 檔案，使用其他檔案類型時，會忽略這些選項。它們對應於標準「開啟檔案」對話框中可用的選項。

要讀取的工作表號碼。 如果檔案具有多個工作表，請依號碼或名稱指定要讀取的工作表。依預設，會讀取 1 號工作表。請勿同時指定名稱和號碼。

要讀取的工作表名稱。 您可以在這裡輸入名稱，以指定要讀取的工作表。如果同時提供名稱和號碼，則名稱優先。

資料格範圍。 依預設，會讀取所有資料格。您可以指定資料格範圍，以限制讀取的區域。

從第一列讀取變數名稱。 勾選此方框，以便從範圍的第一列取得 SPSS 資料集的變數名稱。 系統將會自動取代無效的 SPSS 名稱。

採用的字串寬度上限。 依預設，為字串指派的寬度是 32767。您可以在這裡指定更小的數字。

用於 SAS 檔案的選項。 您可以輸入要讀取的檔案內的特定 SAS 資料集名稱。

其他特性

執行指令
SPSSINC GETURI DATA /HELP.
以顯示整個語法說明。

需求

此指令需要 Python Essentials。您可以從 SPSS 社群網站 (www.ibm.com/developerworks/spssdevcentral) 或安裝資料中下載它們。Essentials 會自動隨 Statistics 22 或更新版本一起安裝。
