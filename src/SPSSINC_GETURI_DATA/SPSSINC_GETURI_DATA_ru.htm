Открыть файл данных по адресу в Интернете

Это диалоговое окно используется, чтобы открыть файл данных в формате SPSS, Excel, SAS или Stata, зная веб-адрес. Этот файл будет скачан на ваш компьютер и после этого открыт. Это диалоговое окно можно использовать для скачивания и сохранения файлов любых других типов объектов, но они не будут открываться.

Положение в сети и имя файла данных. Чтобы открыть файл, введите полный Интернет-адрес, в том числе протокол, такой как http:// или ftp://. Файл будет скачан во временный каталог на вашем компьютере и открыт. Если вы предполагаете использовать этот файл в дальнейшем, сохраните его в постоянном положении.

Тип файла. В качестве типа данных выберите SPSS, Excel, SAS или Stata. Поддерживаемые версии этих файлов соответствуют версиям, которые вы можете открыть в стандартном диалоговом окне Файл - Открыть. Выберите Другой, чтобы только скачать и сохранить файл.

Папка сохранения. Если в качестве типа файла выбрать Другой, укажите существующую папку, где нужно сохранить файл. Имя файла будет взято из URI скачивания.

Существующий файл может быть или перезаписан, или нет в зависимости от операционной системы и разрешений файла. Если возникает противоречие с именами файлов, данный файл может быть оставлен в папке назначения с временным именем.

Имя набора данных для назначения. Дополнительно открываемому файлу можно назначить имя набора данных SPSS. Это не имя файла, но такое имя можно использовать для управления файлами данных, когда можно открывать несколько файлов.

Опции для файлов Excel. Данные опции применимы только к файлам Excel; они игнорируются, если используется другой тип файла. Они соответствуют опциям, доступным в стандартном диалоговом окне Файл - Открыть.

Номер листа для чтения. Если файл содержит несколько листов, укажите, какой из них прочесть, по номеру или имени. По умолчанию читается лист номер 1. Не указывайте одновременно имя и номер.

Имя листа для чтения. Введя здесь имя, можно указать лист для чтения. Если одновременно заданы номер и имя, приоритет имеет имя.

Диапазон ячеек. По умолчанию читаются все ячейки. Чтобы ограничить область чтения, можно задать диапазон ячеек.

Читать имена переменных из первой строки. Включите этот переключатель, чтобы брать имена переменных для набора данных SPSS из первой строки диапазона. Имена, недопустимые в качестве имен SPSS, будут автоматически заменяться.

Предполагаемая максимальная длина строки. По умолчанию длине строк присваивается значение 32767. Здесь можно указать меньшее число.

Опции для файлов SAS. Имя конкретного набора данных SAS можно ввести в файле для чтения.

Дополнительные возможности

Введите команду
SPSSINC GETURI DATA /HELP.
чтобы вывести полное описание синтаксиса.

Технические требования

Для этой команды необходим пакет Python Essentials. Этот пакет можно скачать через веб-сайт сообщества SPSS по адресу www.ibm.com/developerworks/spssdevcentral или взять с установочного носителя. Пакеты Essentials автоматически устанавливаются с версией Statistics 22 или более новой.
